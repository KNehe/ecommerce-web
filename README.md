## Getting Started

- First, run the development server:

```bash
npm run dev
# or
yarn dev
```

- Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

- Or [Visit the live app](https://ecommerce-web-woad.vercel.app/) to see the result
